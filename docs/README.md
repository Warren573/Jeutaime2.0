# JEUTAIME DOCUMENTATION

Ce dossier contient la source officielle des règles produit, UX, gameplay et direction artistique de JeuTaime.

Tous les agents IA (Claude, Codex, Copilot, etc.) doivent lire ces documents avant toute modification importante.

Règles absolues :

* mobile-first uniquement
* jamais de débordement horizontal
* jamais de layout desktop-like
* respecter les safe-area-inset
* conserver la cohérence visuelle
* éviter le style gnangnan/cartoon
* respecter les mécaniques de lettres, matching et salons
* ne pas réintroduire les éléments abandonnés

La documentation est prioritaire sur les suppositions de l’IA.
